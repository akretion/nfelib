This directory contains legacy NFe 4 - 29/06/2021 xsd for legacy bindings
    
This allows transition legacy generateDS nfelib 1.3.1 bindings (nfelib/v4_00)
to validate against legacy NFe 4.0.0 schema for which they where generated:
Pacote de Liberação nº 9 (Novo leiaute da NF-e, NT 2020.005, 2020.006 e
NT 2021.002). Publicado em 29/06/2021.

This avoids validation errors when trying to validate NFe using
the legacy bindings along with the newest schemas.
